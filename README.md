# Proyecto It’s Natt

## Descripción

It’s Natt es una tienda online de indumentaria femenina desarrollada como proyecto práctico para la materia Programación Web I.

El sitio está orientado a la venta y exhibición de prendas y accesorios de moda femenina, incluyendo categorías como vestidos, abrigos, prendas superiores, prendas inferiores, ropa interior y accesorios.

La propuesta visual del proyecto busca transmitir una estética moderna, minimalista y elegante, inspirada en tiendas de moda contemporáneas. A lo largo del sitio se incorporan imágenes de alta calidad, galerías visuales, cards de productos y una guía de talles pensada para mejorar la experiencia del usuario.

Además de la presentación de productos, el proyecto incluye una sección de contacto y una página de talles desarrollada mediante tablas HTML.

## Estructura y etiquetas utilizadas

La estructura general del proyecto fue organizada utilizando etiquetas semánticas como:

- `<html>`
- `<head>`
- `<body>`
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<footer>`

Dentro del documento también se utilizaron etiquetas como `<meta>` para configurar la codificación UTF-8 y el diseño responsive, `<link>` para conectar Google Fonts y el archivo CSS externo, y `<title>` para definir el nombre de cada página.

La navegación del sitio fue desarrollada mediante enlaces `<a>` junto con listas `<ul>` y `<li>`, permitiendo acceder a las distintas páginas del proyecto.

Además, se utilizaron etiquetas `<div>` para agrupar elementos visuales como contenedores, cards y estructuras de diseño.

## Contenido del sitio

La página principal incluye títulos y textos mediante etiquetas:

- `<h1>`
- `<h2>`
- `<h3>`
- `<p>`

También se incorporaron imágenes mediante la etiqueta `<img>`, mostrando distintas categorías y productos de la tienda.

Las páginas de productos fueron diseñadas utilizando cards responsivas, donde cada tarjeta contiene:

- Imagen del producto
- Nombre
- Descripción
- Precio
- Botón interactivo “Agregar al carrito”

## Tabla de talles

El proyecto incorpora una página exclusiva de guía de talles desarrollada mediante tablas HTML utilizando las etiquetas:

- `<table>`
- `<thead>`
- `<tbody>`
- `<tr>`
- `<th>`
- `<td>`

La sección permite visualizar equivalencias de talles y medidas corporales de diferentes prendas.

## Formulario de contacto

El sitio también incluye un formulario de contacto desarrollado mediante:

- `<form>`
- `<input>`
- `<textarea>`
- `<button>`

El formulario permite que los usuarios escriban consultas o mensajes relacionados con la tienda.

## Diseño visual y estilos CSS

El diseño visual fue desarrollado utilizando un archivo CSS externo (`styles.css`), aplicando estilos personalizados para:

- Header y footer
- Navegación
- Formularios
- Cards de productos
- Tablas
- Botones
- Galerías de imágenes

También se utilizaron propiedades CSS como:

- `display: flex`
- `display: grid`
- `gap`
- `padding`
- `margin`
- `border-radius`
- `box-shadow`
- `object-fit`
- `transition`
- `transform`
- `media queries`

Además, se incorporó la tipografía “Poppins” mediante Google Fonts.

## Responsive Design

El sitio cuenta con diseño responsive adaptado a diferentes tamaños de pantalla mediante el uso de:

- Flexbox
- CSS Grid
- Media Queries

Esto permite que la navegación, las cards de productos, las tablas y las distintas secciones se adapten correctamente tanto en desktop como en dispositivos móviles.

## Animaciones y transiciones

Se implementaron transiciones y efectos visuales utilizando CSS, principalmente en:

- Botones
- Imágenes
- Links de navegación
- Cards de productos

Mediante propiedades como:

- `transition`
- `transform`
- `hover`
- `opacity`
